Шрифт Styrene ALC — коммерческий (Commercial Type). Для HTML/PDF положите в эту папку файл начертания Light, например:
  StyreneALC-Light.otf
или подключите семейство через систему (Font Book / Adobe Fonts).

Скрипты ищут файлы с именами, содержащими «styrene», «alc» и «light» (без учёта регистра).
