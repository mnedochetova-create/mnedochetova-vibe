# МН: Каталог впечатлений (Family office)

**Формат:** один файл [`index.html`](./index.html) — «слайды» 16:9 (как широкоэкранный PowerPoint), прокрутка с привязкой к экрану; светлая редакционная вёрстка в духе материалов FO.

**Структура:** слайд **1** — обложка; **2** — полное **содержание** по странам (как в первой версии каталога); **3** — направление **Италия**; **4–6** — **Рим** в доработанном формате (направление → отель Hotel de Russie с фото с сайта, справкой и картой → впечатления). Далее снова подключены **черновые слайды** по другим странам и городам (Комо, Франция, Испания и т.д.) — их можно наполнять по мере готовности, по тому же шаблону, что Рим.

**МН** — блоки для ваших комментариев (правка текста в HTML).

**Изображения:** у Рима — медиа с сайта Rocco Forte; у остальных разделов пока в основном иллюстрации Unsplash, как в исходном каталоге. При `file://` внешние картинки иногда не грузятся — откройте через локальный сервер или положите копии в `assets/`.

**Нумерация:** справа в шапке или в углу — `текущий / всего` (скрипт в конце `index.html`).

**Печать / PDF:** Chrome → Печать → PDF; при необходимости — `playwright`, как в папке предложения по аудиту FO.
